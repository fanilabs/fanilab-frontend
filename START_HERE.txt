Welcome to FaniLab Frontend! 🚀

This is the user interface for the FaniLab blockchain logistics platform.

--------------------------------------------------------------------------------
BEFORE YOU START
--------------------------------------------------------------------------------

You need:
✓ Node.js 18+ installed (check: node --version)
✓ npm or yarn installed (check: npm --version)
✓ Freighter wallet extension (https://www.freighter.app/)
✓ Smart contracts deployed (from SwiftChain-SmartContract folder)

--------------------------------------------------------------------------------
QUICK START (3 STEPS)
--------------------------------------------------------------------------------

1. INSTALL DEPENDENCIES

   Open Command Prompt or PowerShell in this directory and run:

   npm install

2. CONFIGURE ENVIRONMENT

   Copy the example environment file:

   copy .env.example .env.local

   Then edit .env.local and add your deployed smart contract addresses.

3. RUN THE APPLICATION

   Option A - Using npm:
   npm run dev

   Option B - Using the batch script:
   start-dev.bat

   The app will be available at: http://localhost:3000
